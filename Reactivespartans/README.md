# Reactivespartans App

### Technolgy Stack

|Object|Technology|
|--|--|
|Client|React && Redux |
|Server|Node |
|Database|Mongo DB |
|Deploy| AWS|

### Tools

- **IDE**: Visual Studio Code

### Setup and Run

1. Clone the repo
2. install dependencies 
   `npm install`
3. Run the server and client
   ```
   npm run build # client
   npm start # Server
   ```
